# OpenCode Obsidian Plugin

An OpenCode client, inside Obsidian.

> [!WARNING]
> This is a work in progress. Not yet complete or suitable for day-to-day use. Try [this one](https://github.com/mtymek/opencode-obsidian/tree/main#) for a complete working implementation.

OpenCode is a server that can be plugged into various frontends; this plugin provides an Obsidian interface to the OpenCode service.

This project uses TypeScript to provide type checking and documentation.
